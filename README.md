# Comment Order

一个专为开发者设计的 IntelliJ IDEA 插件，用于快速为代码注释添加有序编号或自定义引导符号。

## 🚀 主要功能

- **智能递增**：自动识别选中的注释行，并按顺序添加 `1.`, `2.`, `3.` 等编号。
- **自定义符号**：可在设置中切换为固定符号模式（例如：`// • 注释内容`）。
- **批量处理**：支持一次性选中多行注释或使用多光标（Multi-carets）进行操作。

## ⌨️ 快捷键

选中目标注释行后，按下：

- **`Alt + Shift + Q`** (Windows/Linux/macOS)

## ⚙️ 插件设置

你可以根据个人习惯调整序号样式：

1. 打开 IDE **Settings** (快捷键 `Ctrl+Alt+S`)。
2. 导航至菜单项里  **Comment Order Settings**（“其他设置”中，也可以搜索“Comment Order Settings”）。
3. 在此处你可以：
   - 勾选 **数字序号** 开启 1, 2, 3 模式。
   - 勾选 **自定义字符** 并输入你想要的符号（如 `*`, `•` 等）。

## ⚠️ 注意事项（重要）

1. **仅支持独立注释行**：
   插件目前仅能识别**以注释符开头**（行首允许有空格）的行。
   - ✅ 支持：`// 这是一个步骤`
   - ✅ 支持：`   /* 块注释内容 */`
   - ❌ **不支持**：`var code = 1; // 这种代码后的注释暂不支持`
2. **语言兼容性**：支持所有使用 `//`、`/* */` 或 `*`（多行注释内部）作为注释符的语言（Java, Kotlin, C++, JS 等）。
